Level 01 · Dasar

Fundamental Web

  • HTML5 & Semantic HTML
  • CSS3 & Flexbox/Grid
  • Responsive Design
  • JavaScript Dasar
  • DOM Manipulation
  • Proyek: Landing Page
Level 02 · Menengah

Web Development

  • JavaScript Lanjutan (ES6+)
  • Fetch API & AJAX
  • CSS Framework (Tailwind/Bootstrap)
  • Version Control (Git & GitHub)
  • Deployment (Netlify/Vercel)

Level 03 · Mahir

Framework & Modern Web

  • React.js / Vue.js
  • State Management
  • API Integration
  • Authentication & Authorization
  • Database (Firebase/MySQL)
  • Proyek: Fullstack App
